Sales & Profitability Dashboard
Objective

To build an interactive dashboard visualizing sales performance, profitability, and discount impact.

Tools Used
Google Looker Studio
Data Visualization
Key Insights
Technology leads in both sales and profitability.
Furniture shows strong sales but weak margins due to discounting.
Profitability declines sharply beyond 20% discount.
